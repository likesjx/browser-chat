Getting Started

This is the MVP (Minimum Viable Product) test page for the Browser Chat component.

üöÄ Real ONNX Inference: This version uses real language models running entirely in your browser via ONNX Runtime Web (through transformers.js). The first load will download Gemma 3 270M (~270MB), then it's cached locally for instant subsequent loads.
  1. Press Cmd-K (Mac) or Ctrl-K (Windows/Linux) to activate the chat
  2. Type your message in the input box
  3. Press Enter to submit (use Shift-Enter for new lines)
  4. Watch tokens stream in real-time!
  5. Press Esc to cancel generation or close the chat
